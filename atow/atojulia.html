Julia REPL Basic Command

REPL Command Definition
ans Recall last result
Ctrl + C Interrupt execution
Ctrl + L Clear screen
workspace() Clear all variables and reset session
include("filename.jl") Run program
?func Get help for func is defined
apropos("func") See all places where func is defined
; Escape to OS command line
quit() or Ctnrl + D Exit

  • Julia reaches the speed of compiled languages such as C, C++ and Fortran.

These two features gives Julia the phrase "Walks like Python, Runs like C".


Another Good Features of Julia Programming Lanuage

  • Julia is a multiple dispatch language - In a multiple dispatch language, a function call may dispatch to different functions depending on the type called run-time, which stands in contrast to a single dispatch language, in which a function call always sends to the same function. A multiple dispatch environment ensures that, depending on the variables provided to the function, diffrent results will be produced
  • Julia is great for parallelism - The most amazing feature of Julia is its parallelism. it can parallelize directly from the command line by calling the desired script with a given number of cores. Additionally, it is possible to send a tasks to different threads directly from code. Lastly, Julia can run loops in parallel directly. That is, one does not need to designating different threads to each loop.
